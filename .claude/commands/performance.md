# パフォーマンス分析コマンド

分析対象: $ARGUMENTS

## 分析項目

### メモリ使用量
- Core Dataのメモリ使用状況
- 画像キャッシュのサイズ
- メモリリークの検出

### CPU使用率
- メインスレッドのブロッキング処理
- バックグラウンドタスクの負荷
- アニメーションのフレームレート

### バッテリー消費
- 位置情報更新の頻度と精度
- バックグラウンド処理の最適化
- ネットワーク通信の効率

### ネットワーク
- API呼び出しの頻度
- キャッシュの有効性
- レスポンスサイズ

## 最適化提案
1. 問題のある箇所を特定
2. 具体的な改善コードを提示
3. ビフォー/アフターの比較

Instrumentsで測定すべき項目も提示します。
