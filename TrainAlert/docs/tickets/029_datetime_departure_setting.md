# Ticket #029: 出発日時の詳細設定機能

## 概要
経路設定で出発時刻を日時単位（日付＋時刻）で設定できる機能を実装する。翌日の電車や特定の日付の電車を検索・設定可能にする。

## 実装ガイドライン
- 既存の時刻のみ設定との互換性を保つ
- デフォルトは現在日時
- 未来の日付は最大1ヶ月先まで
- 祝日データはiOSのカレンダーAPIを使用

## 完了条件（Definition of Done）

## テスト方法
1. 当日の時刻設定
2. 翌日の時刻設定
3. 週末/祝日の時刻設定
4. 深夜0時をまたぐ設定
