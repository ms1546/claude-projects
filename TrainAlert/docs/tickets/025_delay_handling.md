# チケット #025: 遅延対応機能

## 概要
リアルタイム遅延情報を取得し、通知時刻を自動調整する機能を実装する。

## 要件
1. **遅延情報取得**
   - ODPT APIからリアルタイム情報を取得
   - 遅延時間を通知に反映

2. **通知時刻調整**
   - 遅延分だけ通知時刻を後ろにずらす
   - 例：10分遅延 → 通知も10分遅らせる

3. **UI表示**
   - 遅延情報の表示
   - 調整後の通知時刻表示

## 技術仕様
### APIエンドポイント
```
GET /api/v4/odpt:Train
?acl:consumerKey={API_KEY}
&odpt:railway={路線ID}
```

### データ構造
```swift
struct TrainInfo {
    let trainNumber: String
    let delay: Int? // 遅延秒数
    let currentStation: String?
}
```
